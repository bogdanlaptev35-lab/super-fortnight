# [cite_start]Инструкция по начальной настройке конфигурации «Управление IT-фирмой» [cite: 1]

[cite_start]Новый функциональный блок разработан для компании «Ваш компьютерный мастер»[cite: 1, 2]. 
[cite_start]Реализованы подсистемы «Работа с заявками клиентов» и «Расчёт управленческой зарплаты», а также добавлены соответствующие отчеты и автотесты[cite: 4, 5, 6].

## 1. Настройка Telegram-бота для оповещений
[cite_start]Для работы системы оповещений специалистов о новых заявках необходимо настроить Telegram-бота[cite: 20]:
1. [cite_start]В Telegram найдите @BotFather и отправьте команду `/start`, затем `/newbot`[cite: 39, 40].
2. [cite_start]Задайте имя бота и его идентификатор (обязательно заканчивается на "bot")[cite: 41, 42].
3. [cite_start]Скопируйте полученный токен и вставьте его в 1С в константу «Токен управления телеграм-ботом»[cite: 43].
4. [cite_start]Создайте группу в Telegram, добавьте туда бота и назначьте его администратором[cite: 44, 45, 46].
5. [cite_start]Напишите в группу любое сообщение и перейдите по ссылке (через браузер или Postman): `https://api.telegram.org/bot[ВашТокен]/getUpdates`[cite: 47].
6. [cite_start]Найдите в полученном JSON идентификатор группы и сохраните его в константу «Идентификатор группы для оповещения»[cite: 48].
7. [cite_start]В 1С настройте интервал отправки для регламентного задания «Отправка уведомлений телеграм боту»[cite: 36, 37].

## 2. Настройка базы для автотестов (Vanessa Automation)
1. [cite_start]Разверните копию базы (укажите, что база является перенесенной)[cite: 119].
2. [cite_start]Снимите безопасный режим для пользователя Администратор (Администрирование – Пользователи)[cite: 122].
3. [cite_start]В параметрах запуска информационной базы укажите `/TESTMANAGER`[cite: 122, 123].
4. [cite_start]Запустите 1С в пользовательском режиме и откройте обработку `vanessa-automation.epf`[cite: 124].
5. [cite_start]На вкладке «Клиенты тестирования» добавьте клиентов с нужными профилями доступа (например: Специалист, БухгалтерИТФирмы, Менеджер)[cite: 125, 126, 127]. [cite_start]В дополнительных параметрах укажите пользователя через `/N"ИмяПользователя"`[cite: 127].
